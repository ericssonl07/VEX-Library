\doxysubsection{Detailed Description}
The {\ttfamily \doxylink{class_virtual_odometry}{Virtual\+Odometry}} class tracks the position and rotation of the robot, in a virtual environment. A "{}virtual"{} environment is a minimal VEX development setup with no physical robot, and only two motors to simulate basic movement. 

\begin{DoxyAttention}{Attention}
The {\ttfamily \doxylink{class_virtual_odometry}{Virtual\+Odometry}} class will only function within scope, so make sure it is managed by another object. By default, {\ttfamily \doxylink{class_virtual_chassis}{Virtual\+Chassis}} instances (and all instances of derived classes) manages the {\ttfamily \doxylink{class_virtual_odometry}{Virtual\+Odometry}} object. 
\end{DoxyAttention}
